## Overview
This project demonstrates ethical job data collection using the RemoteOK public API.

## Data Source
- RemoteOK public JSON API
- Endpoint: https://remoteok.com/api

## Methodology
- Used a public API (no HTML scraping)
- Read-only access
- No authentication required
- No rate abuse

## Output
The project generates:
- `remoteok_jobs.csv`
- `remoteok_jobs.xlsx`

Each file contains:
- Job Title
- Company
- Location

## Tools Used
- Python
- Requests
- Pandas

## Ethics Statement
This project respects ethical data collection principles by using a publicly available API and avoiding scraping of protected or dynamic web content.
